# MindRL MVP Sync for 2026-07-25

## Goal

Build a transparent RL infrastructure MVP for AR LLM and diffusion research,
using the MINDRL interface-controller idea as the cross-branch control layer.

## Completed in the MVP Skeleton

- New repository name and package: `mindrl`.
- Core abstractions for rollout, reward, teacher signal, objective, and report.
- AR LLM objective layer:
  - exact-match reward
  - GRPO-style group-relative objective
  - OPD-style token supervision
  - batch-baseline policy-gradient objective
- Diffusion objective layer:
  - DDPO-style denoising trajectory objective
  - compressibility reward
  - CLIP-like alignment proxy
  - controller-backed diffusion summary
- Smoke reports:
  - `examples/run_mvp_smoke.py`
  - JSONL and Markdown report output

## Not in Scope for Phase 1

- No custom distributed runtime.
- No Ray/vLLM/FSDP replacement.
- No full Agent/VLA/world-model stack in the MVP.
- No confirmed `OBP` algorithm until the term is clarified.

## Next Implementation Targets

- Real AR rollout adapter for Qwen/Llama.
- Teacher-logprob adapter for OPD.
- LoRA/QLoRA training adapter.
- Real diffusers/DDPO adapter that emits `DiffusionTrajectory`.
- Controller ablation across AR and diffusion smoke runs.
